by Robin Schanbacher (robin@schanbros.de)
![App-Icon](images/screenity.png)

# What does it do?
If you install Screenity on all your devices (currently just android possible), it shows you all your screentime  – from all devices.

# What is it able to do?
- collect your screentime and send it to a self-hosted server
- show your today's, local's and each device's screentime
- multiple accounts

# What is planned to implement?
- Widgets
- filters
- admin-page (web)
